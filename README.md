# LibrarySystem (C++)
Simple object-orinted library management system written in C++

<b>Features</b>
Add books to the library
List all books
Interactive terminal menu
Object-Oriented design (Book + Library classes)

<b>Concepts used</b>
Classes and Objects (OOP)
Constructors
Vectors (std::vector)
References (const &)
Loops and user input
Multi-file C++ project structure

<b>Project structure</b>
LibrarySystem/
- main.cpp
- Book.cpp
- Book.h
- Library.cpp
- Library.h
  
-  README.md

<b>How to Run</b>

Compile:
clang++ main.cpp Book.cpp Library.cpp -o app

Run:
./app
